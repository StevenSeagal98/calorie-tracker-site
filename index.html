Calorie Tracker

There are two options to get started with CalorieTracker.

Download The Executable (For Windows)
Just click the "Download" button below and follow the prompts. Windows and Google Drive will throw security warnings. You can ignore them and proceed with the download safely, this is due to the fact that the application is not signed.

Clone the repository and run the application using Python
Visit our Github repository, clone it and follow the instructions in the ReadMe file.
